# 数字标牌产品展示页面 - 产品开发书

## 项目信息

| 项目 | 内容 |
|------|------|
| **项目名称** | 数字标牌产品展示页面 |
| **客户** | 杭州海拓商通国际贸易有限公司 |
| **目标用户** | 广告商、装修公司 |
| **开发周期** | 单次会话完成 |

---

## 一、项目目标

### 1.1 核心目标
创建一个视觉精美、交互流畅的产品展示页面，通过场景化展示让客户直观了解数字标牌产品的应用效果。

### 1.2 成功标准
- ✅ 场景图能够铺满页面，左右切换流畅
- ✅ 产品位置有明显的脉冲闪烁点击点
- ✅ 点击后显示产品详情，带有连线和淡化动画
- ✅ 点击返回或外部区域可回到主场景
- ✅ 整体视觉效果美观专业

---

## 二、功能需求规格

### 2.1 场景浏览功能
| 功能 | 描述 |
|------|------|
| 场景图展示 | 场景图全屏铺满，保持比例自适应窗口 |
| 左右切换 | 页面左右两侧显示三角箭头按钮 |
| 切换动画 | 场景切换时带有平滑过渡效果 |
| 场景顺序 | 按文档中列出的6个场景顺序排列 |

### 2.2 交互热点功能
| 功能 | 描述 |
|------|------|
| 热点样式 | 半透明脉冲波纹动画效果 |
| 热点位置 | 每个场景图中产品位置放置热点 |
| 点击交互 | 鼠标悬停时有视觉反馈，点击触发详情 |

### 2.3 产品详情功能
| 功能 | 描述 |
|------|------|
| 弹窗样式 | 浮动卡片式设计 |
| 连线动画 | 从热点延伸直线连接到弹窗 |
| 背景淡化 | 背景场景图透明度降低（如70%） |
| 详情内容 | 产品名称、产品图片、简要介绍 |
| 返回方式 | 1) 弹窗内返回按钮 2) 点击弹窗外部 |
| 动画效果 | 所有过渡都有平滑渐变动画 |

## 三、技术架构设计

### 3.1 技术栈选择

| 技术 | 用途 | 选型理由 |
|------|------|---------|
| HTML5 | 页面结构 | 标准、兼容性好 |
| CSS3 | 样式与动画 | 原生支持、性能好、无需额外库 |
| JavaScript (ES6+) | 交互逻辑 | 轻量、无需依赖、适合新手 |

### 3.2 文件结构设计

```

├── css/

├── 场景图/           # 场景图片（已有）
├── findings.md       # 需求文档

```

## 四、界面设计规格

### 4.1 主界面布局

```
┌─────────────────────────────────────────┐
│                                         │
│           场景图（全屏）                 │
│                                         │
│  [◀]           热点位置          [▶]        │
│                                         │
│                                         │
└─────────────────────────────────────────┘
```

### 4.2 视觉风格
- **整体风格**：现代、简洁、专业
- **颜色方案**：
  - 主色调：深蓝色系（商务感）
  - 热点色：半透明白色 + 蓝色脉冲
  - 弹窗色：白色卡片 + 浅阴影
- **动画时间**：0.3-0.5秒平滑过渡

### 4.3 热点设计
- 形状：圆形
- 大小：40-50px直径
- 动画：向外扩散的波纹脉冲效果
- 透明度：50-70%半透明

### 4.4 详情弹窗设计
- 位置：浮动卡片，跟随热点附近
- 尺寸：约400x500px
- 内容布局：
  ```
  ┌─────────────────────┐
  │   [返回]   产品名称 │
  ├─────────────────────┤
  │                     │
  │    产品图片区域     │
  │                     │
  ├─────────────────────┤
  │   产品简要介绍      │
  └─────────────────────┘
  ```
